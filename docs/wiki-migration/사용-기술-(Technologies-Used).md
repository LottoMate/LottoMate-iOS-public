## Swift(UIKit)
- 활용 방법: iOS 앱 개발의 주 언어로 사용되며, UIKit 프레임워크를 통해 사용자 인터페이스를 구성합니다.
- 효과: Swift의 간결한 문법과 강력한 타입 시스템 덕분에 안정적이고 유지보수가 쉬운 코드를 작성할 수 있습니다.

## FinLayout & PinLayout
- 활용 방법: 유연한 레이아웃 구성을 위해 함께 사용되며, 다양한 화면 크기에 대응하는 UI를 생성하고 비율 기반 레이아웃을 구현합니다.
- 효과: 다이나믹한 UI 구성으로 다양한 기기에서의 사용자 경험을 향상시키고, UI 요소가 자동으로 조정되어 일관된 디자인을 유지할 수 있습니다.

## RxSwift
- 활용 방법: 비동기 작업 및 데이터 흐름을 관리하기 위해 사용하며, 반응형 프로그래밍 패러다임을 적용합니다.
- 효과: 코드의 가독성과 유지보수성을 높이고, 비동기 이벤트 처리에 따른 복잡성을 줄입니다.

## Moya
- 활용 방법: API 요청을 간편하게 관리하기 위해 사용되며, 네트워킹 코드의 재사용성을 높입니다.
- 효과: API 호출을 모듈화하고 간소화하여, 유지보수와 테스트가 용이해집니다.

## ReactorKit
- 활용 방법: View와 Business Logic 사이의 명확한 분리를 위해 사용되며, 반응형 상태 관리를 제공합니다.
- 효과: 상태 변화와 로직을 명확하게 관리함으로써 코드의 유지보수성을 높이고, 테스트가 용이해집니다. 또한, RxSwift와의 자연스러운 연동으로 비동기 작업 관리에 효율적입니다.
